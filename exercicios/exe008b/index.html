Segundo Elizabeth, no seu livro HTML:

Hoy en día, cualquier persona puede crear su propio sitio Web y luego presentarlo al público de Internet. Todas las páginas Web están escritas con alguna forma HTML, que cuenta con dos características esenciales: el hipertexto y la universalidad. Aunque hay muchos programas de software que le crearán código HTML, usarlo significa no tener que estudiar nuevo software y estar limitado por las características y utilidades de un determinado programa. No es difícil aprender a dominar HTML.
